3. UI 與教學性調整

編修模式

閱讀模式維持乾淨。使用者點「編輯」後，文字框下方會顯示即時預覽，讓新手看見模板文字與變數代入後實際文字的差異。

Voice / OCR

Voice 使用藍紫色系，OCR 使用暖橘色系。兩者都是資料入口，但視覺上明確區分任務性質。

Warm 主題

第三種主題由 Teaching 改為 Warm，採暖色馬卡龍風格，適合課堂展示與初學者使用。
