2. 本版修訂重點

精簡模式切換

主中心頂部新增 Segmented Control 模式切換鈕，預設為精簡模式，藉由 localStorage 記憶使用者最後選取的狀態。

視覺過濾隱藏

在精簡模式下，卡片與編輯器中的次要行政欄位（說明、標籤、預期輸出、人工檢查點、目標 AI 與版本）會自動以 CSS 隱藏，僅保留標題與 Prompt 內容。

編輯安全機制

在精簡模式下編輯並保存卡片，僅更新編輯的標題與核心 Prompt，其他被隱藏的次要欄位值會安全保留，不會遺失。

外掛側邊欄切換卡片變數更新修復

修復了 Chrome 外掛側邊欄在切換/點選不同卡片時，置頂變數面板內容沒有跟著更新的 Bug，保證資料連動即時性。

3. 衛星控制台外掛升級

置頂固定變數即時同步 (Variable Syncing Bug Fix)

修復了舊版本中點選不同 Prompt 卡片時，置頂變數輸入框維持在上一張卡片變數的問題。在清單重繪的尾端自動呼叫變數更新邏輯，確保變數區域隨時與當前展開的卡片內容同步。

版本號全面升級為 v2.2

外掛版本顯示同步升級為 v2.2（對應 manifest.json 版本 2.2.0），與主中心保持一致。
